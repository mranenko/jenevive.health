# jenevive.health


## 1. Create

In the project (`axon`) directory run:

```bash
npm create vite@latest .
```

Choose Vanilla JS.

---


## 2. Install

```bash
npm install --save-dev sass
npm install @phosphor-icons/web
npm install
```

---


## 3. Run

```bash
npm start
```
